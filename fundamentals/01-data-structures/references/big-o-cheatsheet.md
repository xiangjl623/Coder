# 大 O 速查表

## 常见复杂度排序（慢 → 快）

O(n!) > O(2^n) > O(n²) > O(n log n) > O(n) > O(log n) > O(1)

## 算法复杂度

| 算法 | 最好 | 平均 | 最坏 | 空间 |
|------|------|------|------|------|
| 冒泡排序 | O(n) | O(n²) | O(n²) | O(1) |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) |
| 插入排序 | O(n) | O(n²) | O(n²) | O(1) |
| 快速排序 | O(n log n) | O(n log n) | O(n²) | O(log n) |
| 线性查找 | O(1) | O(n) | O(n) | O(1) |
| 二分查找 | O(1) | O(log n) | O(log n) | O(1) |
