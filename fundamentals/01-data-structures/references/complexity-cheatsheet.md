# 数据结构复杂度速查

| 结构 | 访问 | 查找 | 插入 | 删除 | 备注 |
|------|------|------|------|------|------|
| 数组 | O(1) | O(n) | O(n) | O(n) | 末尾追加均摊 O(1) |
| 单向链表 | O(n) | O(n) | O(1)* | O(1)* | *已知前驱节点 |
| 栈（数组） | 栈顶 O(1) | — | O(1) | O(1) | |
| 队列（环形数组） | — | — | O(1) | O(1) | |
| 二叉搜索树 | O(log n)~O(n) | O(log n)~O(n) | 同左 | 同左 | 退化链表时为 O(n) |
| 哈希表 | — | O(1) 均摊 | O(1) 均摊 | O(1) 均摊 | 最坏 O(n) |

## 选型建议

| 需求 | 推荐 |
|------|------|
| 固定大小、下标访问 | 数组 |
| 频繁头插、动态长度 | 链表 |
| 撤销/括号匹配 | 栈 |
| 任务排队、BFS | 队列 |
| 有序查找 | BST 或有序数组 + 二分 |
