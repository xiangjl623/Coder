## 冒泡排序（Bubble Sort）

相邻元素比较，较大者后移，每轮将最大值「冒泡」到末尾。

```c
void bubble_sort(int a[], int n) {
    for (int i = 0; i < n - 1; i++)
        for (int j = 0; j < n - 1 - i; j++)
            if (a[j] > a[j + 1]) {
                int t = a[j];
                a[j] = a[j + 1];
                a[j + 1] = t;
            }
}
```

- 时间：O(n²)，最好 O(n)（已有序且加标志位提前退出）
- 空间：O(1)
- 稳定：是

## 选择排序（Selection Sort）

每轮选最小元素放到前面。时间 O(n²)，不稳定。

## 插入排序（Insertion Sort）

将元素插入已排序部分的正确位置。近乎有序时接近 O(n)。

## 快速排序（Quick Sort）概览

选基准 pivot，分区：小于 pivot 放左，大于放右，递归左右子数组。

- 平均 O(n log n)，最坏 O(n²)
- 工程中最常用的比较排序之一

## 算法对比

| 算法 | 平均时间 | 空间 | 稳定 | 考试频率 |
|------|----------|------|------|----------|
| 冒泡 | O(n²) | O(1) | 是 | 高 |
| 选择 | O(n²) | O(1) | 否 | 中 |
| 插入 | O(n²) | O(1) | 是 | 中 |
| 快排 | O(n log n) | O(log n) 栈 | 否 | 中 |

## 本节小结

- 期末重点：冒泡排序手写、理解双重循环与 O(n²)
- 稳定性：相等元素相对顺序是否改变
- 选型：小数据或教学用简单排序；大数据用 O(n log n) 算法
