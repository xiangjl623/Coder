## 推荐复习顺序

1. 阅读 [期末考试大纲.md](../../../languages/c/exams/期末考试大纲.md) 了解全局
2. 按 [两周复习计划.md](../../../languages/c/exams/两周复习计划.md) 执行每日任务
3. 薄弱章节回看 [languages/c/guides/](../../../languages/c/guides/) 对应教程
4. 刷 [languages/c/exams/](../../../languages/c/exams/) 各题型并对照参考答案
