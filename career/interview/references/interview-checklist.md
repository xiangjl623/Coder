# 面试检查清单

## 面试前（T-1 天）

- [ ] 确认时间、形式（现场 / 视频 / 电话）、联系人
- [ ] 复习 JD 关键词，准备匹配的项目故事（STAR）
- [ ] 准备 1 分钟 / 3 分钟自我介绍
- [ ] 浏览 [01 面试准备](../guides/01-interview-preparation.md) 复盘笔记
- [ ] 测试设备：摄像头、麦克风、网络（视频面）
- [ ] 准备 2～3 个问面试官的问题

## 面试中

- [ ] 听清题意，必要时复述并确认
- [ ] 技术题：是什么 → 原理 → 注意点 → 思路
- [ ] 算法题：举例 → 说复杂度 → 写码 → 自测边界
- [ ] 设计题：先需求与估算，再画图
- [ ] 不会的题：说明思考路径，不要沉默

## 面试后（24h 内）

- [ ] 记录题目与卡壳点
- [ ] 补知识点并链接到对应 guide
- [ ] 发感谢信（可选，外企/部分国内公司）
