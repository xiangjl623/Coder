# 现代栈术语速查

| 术语 | 一句话 |
|------|--------|
| REST | 用 HTTP 动词操作资源的 API 风格 |
| gRPC | HTTP/2 + Protobuf 的高性能 RPC |
| JWT | JSON Web Token，无状态认证载体 |
| OAuth2 | 第三方授权框架，非登录协议本身 |
| API Gateway | 统一入口：路由、鉴权、限流 |
| 幂等 | 同一请求多次执行结果相同 |
| CAP | 分区下一致性与可用性的权衡 |
| 最终一致 | 允许短暂不一致，Eventually 收敛 |
| MQ | 消息队列，解耦/削峰/异步 |
| Redis | 内存 KV，缓存/锁/计数 |
| CDN | 边缘缓存静态资源 |
| 读写分离 | 主库写、从库读 |
| 分库分表 | 水平拆分数据 |
| Docker | 容器运行时，镜像打包环境 |
| K8s | 容器编排：Deployment/Service/Pod |
| CI/CD | 持续集成与持续部署 |
| 可观测性 | 日志 + 指标 + 分布式链路 |
| 熔断 | 下游故障时快速失败防雪崩 |
| 限流 | 控制请求速率（令牌桶等） |
| B+ 树 | DB 索引常用结构，叶子链表 |
| MVCC | 多版本并发控制，读写不互斥 |

详见 [04 系统设计与工程实践](../guides/04-system-design-and-engineering.md)。
