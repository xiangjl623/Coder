## 一、判断题（10题）
1. 假设有`int a[10], *p;`，则`p=&a[0]`与`p=a`等价。
2. 若有定义：`int a[2][3];`，则`a[2][10]`能够正确访问`a`数组元素`a[1][1]`。
3. 结构体定义：
```c
struct S
{
    int g; char h;
}T;
```
则可以用`T`来定义结构体变量。
4. 通过`return`语句，函数可以带回一个或一个以上的返回值。
5. `int`类型变量占用4个字节，若有定义：`int x[10]={0,2,4};`，则数组`x`在内存中所占字节数是12字节。
6. 若有`int i=10, j=0;`执行语句：
```c
if(j=0) i++;
else i--;
```
则`i`的值为11。
7. 若有定义语句：`char m[]="china";`，则一维数组的定义中，中括号`[]`省略的值为5。
8. 函数调用语句：`func(rec1,rec2+rec3,(rec4,rec5));`中，含有的实参个数是5。
9. `int a[3][4]={{1},{5},{9}};` 它的作用是将数组各行第一列的元素赋初值，其余元素值为0。
10. C语言本身不提供输入输出语句，输入和输出操作是由函数来实现的。

## 三、程序改错题答案
### 改错1 修改点
1. `void fun(int n)` → `int fun(int n)`
2. `for(i=1;i<n;i++)` → `for(i=2;i<n;i++)`
3. `return m;` → `return k;`
4. 第四处FOUND行无需改动，函数外补充声明`int fun(int n);`

完整修正fun函数：
```c
int fun(int n)
{
    int i,k=1;
    if(n<=1) k=0;
    for(i=2;i<n;i++)
        if(n%i==0) k=0;
    return k;
}
```

### 改错2 修改点
1. `fun()` → `float fun()`
2. `while(fabs(t)<=1e-4)` → `while(fabs(t) > 1e-4)`
3. `i=i+1;` → `i=i+2;`
4. `s=t;` → `s += t;`；`printf("pi=%d\n",s*4);` → `printf("pi=%f\n",s*4);`

完整修正fun函数：
```c
float fun()
{
    int i=1;
    float s=0.0,t=1.0,p=1;
    while(fabs(t) > 1e-4)
    {
        s += t;
        p=-p;
        i=i+2;
        t=p/i;
    }
    printf("pi=%f\n",s*4);
}
```

```c
int fun(int x[],int n)
{

    for(int i=1; i<n; i++)

        if(x[i] < min)
            min = x[i];
    }
    return min;
}
```
