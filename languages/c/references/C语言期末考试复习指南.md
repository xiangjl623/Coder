# 1 C语言程序基础

## ***大纲***

1. **程序入口**：C语言程序必须从`main()`函数开始执行，且有且仅有一个主函数
2. **注释方式**：
   - 多行注释：`/* 内容 */`
   - 单行注释：`// 内容`
3. **语句结束**：每条执行语句必须以英文分号`;`结尾
4. **程序结构**：头文件引入 → 全局定义 → main函数 → 执行语句 → return返回
5. **基本输入输出**：使用`printf()`输出，需引入`stdio.h`头文件
6. **大小写敏感**：C语言严格区分大小写字母
7. **字符串表示**：字符串必须用英文双引号括起来

## 1.1 知识点总结

### 1.1.1 最小程序结构

```c
#include <stdio.h>          /* 预处理：包含头文件 */

int main(void) {            /* 主函数，程序入口 */
    printf("Hello, C!\n");
    return 0;               /* 返回 0 表示正常结束 */
}
```

### 1.1.2 运行流程


| 阶段   | 说明                         | 常见产物               |
| ---- | -------------------------- | ------------------ |
| 预处理  | 处理`#` 指令（include、define 等） | `.i` 文件            |
| 编译   | 将 C 代码翻译为汇编                | `.s` 文件            |
| 汇编   | 汇编代码转为机器码                  | `.o` / `.obj` 目标文件 |
| 链接   | 合并目标文件与库，生成可执行文件           | `.exe` / 无后缀可执行文件  |

### 1.1.3 源文件与头文件


| 类型   | 扩展名  | 作用                      |
| ---- | ---- | ----------------------- |
| 源文件  | `.c` | 存放函数实现                  |
| 头文件  | `.h` | 存放声明、宏、类型定义，供`#include` |

### 1.1.4 注释


| 方式   | 语法             | 说明      |
| ---- | -------------- | ------- |
| 单行   | `// 注释`        | C99 起支持 |
| 单行   | `/* 注释 */`     | 传统写法    |
| 多行   | `/* 多行\n注释 */` | 不能嵌套    |

### 1.1.5 命名规则 (重点)


| 规则     | 说明               | 合法示例            | 非法示例   |
| ------ | ---------------- | --------------- | ------ |
| 首字符    | 字母或`_`           | `_count`, `sum` | `2abc` |
| 后续字符   | 字母、数字、`_`        | `var_1`         | `a-b`  |
| 区分大小写  | `Sum` 与 `sum` 不同 | —               | —      |
| 不能是关键字 | 如`int`、`return`  | `my_int`        | `int`  |

### 1.1.6 关键字（常用）


| 类别    | 关键字                                      |
| ----- | ---------------------------------------- |
| 类型    | `char` `int` `float` `double` `void` `short` `long` `signed` `unsigned` |
| 控制流   | `if` `else` `switch` `case` `default` `for` `while` `do` `break` `continue` `goto` |
| 函数/存储 | `return` `static` `extern` `register` `auto` `const` `volatile` |
| 结构    | `struct` `union` `enum` `typedef`        |
| 其他    | `sizeof` `break` `continue`              |

### 1.1.7 表达式 (重点)


| 概念   | 说明           | 示例                       |
| ---- | ------------ | ------------------------ |
| 表达式  | 有值的式子        | `a + 1`、`x > 0`          |
| 语句   | 以`;` 结尾，执行动作 | `a = 5;`、`printf("hi");` |
| 复合语句 | 用`{}` 包裹多条语句 | `if (x) { a=1; b=2; }`   |

### 1.1.8 主要特点（速记）


| 特点      | 说明             |
| ------- | -------------- |
| 面向过程    | 以函数为组织单位       |
| 编译型     | 先编译再运行，效率高     |
| 贴近硬件    | 可指针操作、位运算      |
| 可移植     | 标准 C 代码跨平台性好   |
| 无自动内存管理 | 动态内存需手动`free`  |
| 弱类型检查   | 数组不检查边界，需程序员注意 |

# 2 数据类型与常量

## ***大纲***

1. **三大基本数据类型**：
   - 整型：`int`
   - 浮点型：`float`（单精度）、`double`（双精度）
   - 字符型：`char`（占用1字节）
2. **常量定义**：使用`#define`预处理指令定义常量，如`#define PI 3.14159`
3. **标识符规则**：
   - 只能由字母、数字、下划线组成
   - 不能以数字开头
   - 不能使用关键字
4. **运算符**：
   - `/`：除法（整数相除取整）
   - `%`：取余（仅适用于整型）
5. **格式化输出**：
   - `%d`：整型格式符
   - `%f`：浮点格式符
   - `%c`：字符格式符

# 3 运算符

## ***大纲***

1. **取余运算符**：`%`仅适用于整型数据，返回除法的余数
2. **自增运算符**：
   - `i++`：后置自增，先使用当前值，再自增1
   - `++i`：前置自增，先自增1，再使用新值
3. **复合赋值运算符**：
   - `a += b` → `a = a + b`
   - `a -= b` → `a = a - b`
   - `a *= b` → `a = a * b`
   - `a /= b` → `a = a / b`
   - `a %= b` → `a = a % b`
4. **运算符优先级**：
   - 最高：`()`（括号）
   - 算术运算符：`*`、`/`、`%` 高于 `+`、`-`
   - 赋值运算符优先级最低
5. **格式化输入**：`scanf("%lf", &r)` 用于读取double类型数据

## 3.1 知识点总结

### 3.1.1 算术运算符（重点）


| 运算符  | 含义   | 示例      | 注意             |
| ---- | ---- | ------- | -------------- |
| `+`  | 加    | `a + b` | —              |
| `-`  | 减    | `a - b` | —              |
| `*`  | 乘    | `a * b` | —              |
| `/`  | 除    | `a / b` | 整数/整数 → 整数（截断） |
| `%`  | 取模   | `a % b` | 仅用于整型          |

| 取地址`&` | 获取变量地址     |
| 解引用`*` | 通过指针访问所指对象 |

### 7.1.2 指针声明与使用


| 写法               | 含义            |
| ---------------- | ------------- |
| `int *p;`        | p 是指向 int 的指针 |
| `p = &x;`        | p 指向 x        |
| `*p = 10;`       | 通过 p 修改 x 的值  |
| `int *p = NULL;` | 空指针，不指向有效对象   |

```c
int x = 5;
int *p = &x;
printf("%d\n", *p);   /* 输出 5 */
*p = 10;              /* x 变为 10 */
```

### 7.1.3 指针与数组（重点）


| 关系     | 说明                         |
| ------ | -------------------------- |
| 数组名    | 多数情况下等价于首元素地址              |
| `a[i]` | 等价于`*(a + i)`              |
| `p[i]` | 等价于`*(p + i)`              |
| 指针移动   | `p+1` 移动一个**元素**大小，不是 1 字节 |

### 7.1.4 指针运算


| 运算         | 说明             |

| `p - n`    | 向前移 n 个元素      |
| `p1 - p2`  | 两指针间元素个数（同数组内） |
| `p1 == p2` | 比较地址是否相同       |

### 7.1.5 常见指针类型


| 类型               | 说明          |
| ---------------- | ----------- |
| `int *p`         | 指向 int      |
| `char *p`        | 指向字符，常用于字符串 |

### 7.1.6 指针作为函数参数


| 用途     | 示例                           |
| ------ | ---------------------------- |

| 返回多个结果 | 通过指针参数写回                     |

### 7.1.7 指针与 const （难点）

# 8 数组

## ***大纲***

1. **数组特性**：
   - 下标从0开始
   - 连续内存存储
   - 数组名代表首元素地址
2. **二维数组**：
   - 行为主序存储（按行存放）

   - 时间复杂度：O(n²)
5. **函数参数**：数组作为参数传递时，实际传递的是首地址

### 8.1.1 一维数组

```c
#include <stdio.h>
void bubbleSort(int a[],int n)
{
    for(int i=0;i<n-1;i++)
        for(int j=0;j<n-i-1;j++)
            if(a[j]>a[j+1])
            {
                int t=a[j];a[j]=a[j+1];a[j+1]=t;
            }
}
int main()
{
    int nums[5]={5,2,9,1,3};
    bubbleSort(nums,5);
    for(int i=0;i<5;i++) printf("%d ",nums[i]);
    return 0;
}
```

### 11.1.2 fopen 打开模式


| 模式              | 含义    | 文件不存在 | 文件存在     |
| --------------- | ----- | ----- | -------- |

| `clearerr(fp)` | 清除错误和 EOF 标志 |

### 11.1.9 文本文件 vs 二进制文件


|      | 文本模式        | 二进制模式         |
| ---- | ----------- | ------------- |
| 打开   | `"r"` `"w"` | `"rb"` `"wb"` |
| 内容   | 人类可读        | 原始字节          |
| 换行   | 可能转换        | 不转换           |
| 适用   | 配置文件、日志     | 结构体、图片、音频     |

### 11.1.10 完整示例：复制文件

```c
#include <stdio.h>
int main(void) {
    FILE *in = fopen("src.txt", "r");
    FILE *out = fopen("dst.txt", "w");
    if (!in || !out) return 1;

    int ch;
    while ((ch = fgetc(in)) != EOF)
        fputc(ch, out);

    fclose(in);
    fclose(out);
    return 0;
}
```

### 11.1.11 完整示例：结构体存盘

```c
typedef struct { int id; char name[20]; } Record;
Record r = {1, "Tom"};
FILE *fp = fopen("data.bin", "wb");
fwrite(&r, sizeof(Record), 1, fp);
fclose(fp);
```

### 11.1.12 文件操作注意事项


| 要点        | 说明                                       |
| --------- | ---------------------------------------- |
| 检查 fopen  | 失败时返回 NULL                               |
| 必须 fclose | 否则可能丢数据、泄漏资源                             |
| w 模式      | 会清空已有文件                                  |
| 路径        | Windows 可用`"C:\\\\data.txt"` 或 `"C:/data.txt"` |
| 权限        | 确保有读写权限                                  |

# 12 预处理

## 12.1 处理指令概览


| 指令                           | 作用          |
| ---------------------------- | ----------- |
| `#include`                   | 包含头文件       |
| `#define`                    | 宏定义         |
| `#undef`                     | 取消宏定义       |
| `#if` / `#ifdef` / `#ifndef` | 条件编译        |
| `#else` / `#elif` / `#endif` | 条件分支        |
| `#pragma`                    | 编译器相关（实现依赖） |
| `#error`                     | 编译报错并输出信息   |

**特点：** 以 `#` 开头，**不需要**分号结尾。

## 12.2 文件包含


| 写法                    | 搜索顺序        |
| --------------------- | ----------- |
| `#include <stdio.h>`  | 系统目录（标准库）   |
| `#include "myfile.h"` | 当前目录 → 系统目录 |

**常见标准头文件：**


| 头文件         | 内容           |
| ----------- | ------------ |
| `stdio.h`   | 输入输出         |
| `stdlib.h`  | 内存分配、转换、exit |
| `string.h`  | 字符串函数        |
| `math.h`    | 数学函数         |
| `ctype.h`   | 字符处理         |
| `time.h`    | 时间日期         |
| `stdbool.h` | bool 类型（C99） |

## 12.3 定义 #define

### 12.3.1 常量宏

```c
#define PI 3.14159
#define MAX 100
```

### 12.3.2 带参宏

```c
#define MAX(a, b) ((a) > (b) ? (a) : (b))
#define SQUARE(x) ((x) * (x))
```


| 注意    | 说明                                    |
| ----- | ------------------------------------- |
| 加括号   | 防止运算符优先级问题                            |
| 副作用   | 避免`#define SQ(x) x*x` 在 `SQ(i++)` 时出错 |
| 无类型检查 | 宏只是文本替换                               |

### 12.3.3 宏 vs const


|      | `#define` | `const` |
| ---- | --------- | ------- |
| 处理阶段 | 预处理       | 编译      |
| 类型   | 无         | 有       |
| 调试   | 难         | 易       |
| 推荐   | 条件编译、简单常量 | 一般常量    |

## 12.4 件编译

```c
#define DEBUG 1

#ifdef DEBUG
    printf("调试信息\n");
#endif

#ifndef HEADER_H
#define HEADER_H
/* 头文件内容 */
#endif
```


| 指令              | 含义             |
| --------------- | -------------- |
| `#ifdef MACRO`  | 宏已定义则编译        |
| `#ifndef MACRO` | 宏未定义则编译（头文件保护） |
| `#if 表达式`       | 表达式非 0 则编译     |
| `#else`         | 否则分支           |
| `#endif`        | 结束条件块          |

## 12.5 用预处理技巧


| 场景    | 写法                                       |
| ----- | ---------------------------------------- |
| 头文件保护 | `#ifndef XXX_H` / `#define XXX_H` / `#endif` |
| 调试开关  | `#ifdef DEBUG` ... `#endif`              |
| 跨平台   | `#ifdef _WIN32` ... `#else` ...          |
| 字符串化  | `#define STR(x) #x` → `STR(hello)` → `"hello"` |
| 连接符号  | `#define CAT(a,b) a##b`                  |
