# 11 输入输出

## 标准流

| 流 | 文件指针 | 设备 |
|----|----------|------|
| 标准输入 | `stdin` | 键盘 |
| 标准输出 | `stdout` | 屏幕 |
| 标准错误 | `stderr` | 屏幕 |

头文件：`#include <stdio.h>`

## printf 格式输出

### 基本用法

```c
printf("格式字符串", 参数1, 参数2, ...);
```

### 常用格式符

| 格式符 | 类型 | 示例 |
|--------|------|------|
| `%d` / `%i` | int | `printf("%d", 42);` |
| `%u` | unsigned int | `printf("%u", n);` |
| `%ld` / `%lld` | long / long long | `printf("%lld", x);` |
| `%f` | float/double | `printf("%.2f", 3.14159);` |
| `%e` / `%g` | 科学计数法 | `printf("%e", 1.2e3);` |
| `%c` | char | `printf("%c", 'A');` |
| `%s` | 字符串 | `printf("%s", "hello");` |
| `%p` | 指针地址 | `printf("%p", p);` |
| `%x` / `%o` | 十六进制/八进制 | `printf("%x", 255);` |
| `%%` | 输出 `%` | `printf("100%%");` |

### 宽度和精度

| 写法 | 含义 | 示例 |
|------|------|------|
| `%5d` | 宽度至少 5 | 右对齐 |
| `%-5d` | 左对齐 | — |
| `%05d` | 不足补 0 | `00123` |
| `%.2f` | 保留 2 位小数 | `3.14` |
| `%8.2f` | 总宽 8，小数 2 位 | — |

## scanf 格式输入

```c
scanf("格式字符串", &变量1, &变量2);
```

| 格式符 | 类型 | 注意 |
|--------|------|------|
| `%d` | int | 需传地址 `&x` |
| `%lf` | double | double 用 %lf |
| `%f` | float | — |
| `%c` | char | 注意缓冲区残留 `\n` |
| `%s` | 字符串 | 遇空白停止，需数组名（已是地址） |
| `%99s` | 限制宽度 | 防止溢出 |

### 常见陷阱

| 问题 | 解决 |
|------|------|
| 读 char 前残留 `\n` | `scanf(" %c", &ch);` 格式前加空格 |
| 返回值未检查 | `if (scanf("%d", &n) != 1) ...` |
| 数组不需 & | `scanf("%s", s)` 不是 `&s` |

## 字符 I/O

| 函数 | 功能 | 返回值 |
|------|------|--------|
| `getchar()` | 读一个字符 | int（含 EOF） |
| `putchar(c)` | 写一个字符 | 写入字符 |
| `gets` | 读一行 | **已废弃，勿用** |
| `fgets(s,n,stdin)` | 安全读一行 | s |

## 格式化字符串函数

| 函数 | 作用 |
|------|------|
| `sprintf(buf, fmt, ...)` | 格式化写入字符串 |
| `sscanf(s, fmt, ...)` | 从字符串解析 |
| `snprintf(buf, size, fmt, ...)` | 带长度限制，更安全 |

## 文件 I/O 预览（详见 12-文件操作）

| 函数 | 作用 |
|------|------|
| `fopen` / `fclose` | 打开/关闭 |
| `fprintf` / `fscanf` | 文件版 printf/scanf |
| `fgetc` / `fputc` | 字符读写 |
| `fgets` / `fputs` | 行读写 |

## 输入输出示例

```c
#include <stdio.h>
int main(void) {
    int age;
    float height;
    char name[50];

    printf("请输入姓名 年龄 身高: ");
    scanf("%s %d %f", name, &age, &height);
    printf("姓名:%s 年龄:%d 身高:%.2f\n", name, age, height);

    return 0;
}
```

## 缓冲区与 fflush

| 概念 | 说明 |
|------|------|
| 缓冲 | 输出可能暂存缓冲区 |
| `fflush(stdout)` | 强制刷新标准输出 |
| `\n` | 通常触发 stdout 刷新 |

## EOF

| 宏 | 值 | 用途 |
|----|-----|------|
| `EOF` | 通常 -1 | 表示文件结束或读错误 |

```c
int ch;
while ((ch = getchar()) != EOF)
    putchar(ch);
```
