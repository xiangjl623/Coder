# 05 函数

## 函数基本结构

```c
返回类型 函数名(参数列表) {
    /* 函数体 */
    return 返回值;   /* void 函数可省略或写 return; */
}
```

## 函数声明与定义

| 概念 | 说明 | 示例 |
|------|------|------|
| 声明（原型） | 告诉编译器函数存在 | `int max(int a, int b);` |
| 定义 | 函数的具体实现 | 含函数体 |
| 调用 | 使用函数 | `int m = max(3, 5);` |

## 参数传递

| 方式 | 语法 | 特点 |
|------|------|------|
| 值传递 | `void f(int x)` | 修改形参不影响实参 |
| 地址传递 | `void f(int *x)` | 可通过指针修改实参 |
| 数组传递 | `void f(int a[], int n)` | 实际传首元素地址 |

```c
void swap(int *a, int *b) {
    int t = *a; *a = *b; *b = t;
}
/* 调用：swap(&x, &y); */
```

## 返回值

| 返回类型 | 说明 |
|----------|------|
| `int`、`double` 等 | `return 表达式;` |
| `void` | 无返回值，`return;` 或直接结束 |
| 指针 | 可返回指针，**不要返回局部变量地址** |

## 递归函数

| 要素 | 说明 |
|------|------|
| 终止条件 | 必须存在，否则无限递归 |
| 递归关系 | 大问题化为小问题 |

```c
int fact(int n) {
    if (n <= 1) return 1;          /* 终止条件 */
    return n * fact(n - 1);        /* 递归 */
}
```

## 存储类型

| 关键字 | 作用 | 作用域 | 生命周期 |
|--------|------|--------|----------|
| `auto` | 局部变量默认 | 块内 | 函数调用期间 |
| `static`（局部） | 保留值，不重新初始化 | 块内 | 整个程序 |
| `static`（全局） | 仅本文件可见 | 文件内 | 整个程序 |
| `extern` | 声明外部定义的变量/函数 | 跨文件 | — |
| `register` | 建议放寄存器（编译器可忽略） | 块内 | 函数调用期间 |

## 函数分类速查

| 类型 | 示例 |
|------|------|
| 无参无返回值 | `void menu(void);` |
| 有参有返回值 | `int add(int a, int b);` |
| 数组作参数 | `void sort(int a[], int n);` |
| 字符串作参数 | `int len(char s[]);` |
| 函数指针参数 | `void qsort(..., int (*cmp)(...));` |

## main 函数

| 形式 | 说明 |
|------|------|
| `int main(void)` | 无命令行参数 |
| `int main(int argc, char *argv[])` | 接收命令行参数 |
| 返回值 | `return 0;` 表示成功，非 0 表示异常 |

## 头文件与多文件

| 文件 | 内容 |
|------|------|
| `xxx.h` | 函数声明、宏、类型定义 |
| `xxx.c` | 函数实现 |
| 使用 | `#include "xxx.h"` |

**示例：**

```c
/* math_utils.h */
int add(int a, int b);

/* math_utils.c */
int add(int a, int b) { return a + b; }

/* main.c */
#include "math_utils.h"
int main(void) { printf("%d\n", add(1, 2)); return 0; }
```

## 内联函数（C99）

```c
inline int square(int x) { return x * x; }
```

编译器可能将函数体直接展开，减少调用开销。
