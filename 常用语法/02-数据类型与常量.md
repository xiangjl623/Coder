# 02 数据类型与常量

## 基本数据类型

| 类型 | 含义 | 典型大小（32/64位） | 格式符 |
|------|------|---------------------|--------|
| `char` | 字符/小整数 | 1 字节 | `%c` |
| `short` | 短整型 | 2 字节 | `%hd` |
| `int` | 整型 | 4 字节 | `%d` |
| `long` | 长整型 | 4/8 字节 | `%ld` |
| `long long` | 更长整型 | 8 字节 | `%lld` |
| `float` | 单精度浮点 | 4 字节 | `%f` |
| `double` | 双精度浮点 | 8 字节 | `%lf`（scanf）/`%f`（printf） |
| `void` | 无类型 | — | 用于函数返回值或通用指针 |

## 类型修饰符

| 修饰符 | 作用 | 示例 |
|--------|------|------|
| `signed` | 有符号（默认） | `signed int x;` |
| `unsigned` | 无符号，只表示非负数 | `unsigned int n;` |
| `short` | 缩短整型 | `short int s;` |
| `long` | 加长整型 | `long int l;` |
| `const` | 只读，不可修改 | `const int MAX = 100;` |

## 变量声明与初始化

| 写法 | 说明 | 示例 |
|------|------|------|
| 声明 | 指定类型和名字 | `int age;` |
| 初始化 | 定义时赋初值 | `int x = 10;` |
| 多变量 | 同类型一次声明 | `int a = 1, b = 2;` |
| 局部变量 | 未初始化值不确定 | `int x;` |
| 全局/静态 | 未初始化默认为 0 | `static int count;` |

## 整型常量写法

| 进制 | 前缀 | 示例 | 实际值 |
|------|------|------|--------|
| 十进制 | 无 | `255` | 255 |
| 八进制 | `0` | `0377` | 255 |
| 十六进制 | `0x` / `0X` | `0xFF` | 255 |
| 长整型后缀 | `L` / `l` | `100L` | — |
| 无符号后缀 | `U` / `u` | `100U` | — |

## 浮点常量

| 写法 | 说明 | 示例 |
|------|------|------|
| 小数形式 | 含小数点 | `3.14` |
| 指数形式 | e/E 表示 10 的幂 | `1.2e-3` → 0.0012 |
| 单精度后缀 | `f` / `F` | `3.14f` |
| 长双精度后缀 | `L` | `3.14L` |

## 字符与字符串常量

| 类型 | 定界符 | 示例 | 说明 |
|------|--------|------|------|
| 字符 | 单引号 `' '` | `'A'`、`'\n'` | 占 1 字节（存储） |
| 字符串 | 双引号 `" "` | `"Hello"` | 末尾自动加 `\0` |

## 常用转义字符

| 转义 | 含义 |
|------|------|
| `\n` | 换行 |
| `\t` | 制表符 |
| `\\` | 反斜杠 |
| `\'` | 单引号 |
| `\"` | 双引号 |
| `\0` | 空字符（字符串结束） |
| `\ddd` | 八进制 ASCII 码 |
| `\xhh` | 十六进制 ASCII 码 |

## 常量定义方式

| 方式 | 语法 | 特点 |
|------|------|------|
| 宏常量 | `#define PI 3.14` | 预处理替换，无类型 |
| const 变量 | `const double PI = 3.14;` | 有类型，更安全 |
| 枚举 | `enum { RED, GREEN, BLUE };` | 整型常量集 |

## sizeof 求类型/变量大小

| 用法 | 示例 | 结果（常见） |
|------|------|--------------|
| 类型 | `sizeof(int)` | 4 |
| 变量 | `sizeof(x)` | 取决于 x 的类型 |
| 字符 | `sizeof(char)` | 1 |
| 数组 | `sizeof(a)` | 元素个数 × 元素大小 |

## 类型转换

| 类型 | 说明 | 示例 |
|------|------|------|
| 隐式转换 | 编译器自动转换 | `int x = 3.9;` → x=3 |
| 显式转换 | 强制类型转换 | `(int)3.9`、`(float)n` |
| 整型提升 | char/short 运算时提升为 int | `'a' + 1` |

## 取值范围（典型，有符号 int）

| 类型 | 约略范围 |
|------|----------|
| `char` | -128 ~ 127 或 0 ~ 255（unsigned） |
| `short` | -32768 ~ 32767 |
| `int` | 约 ±21 亿 |
| `unsigned int` | 0 ~ 约 42 亿 |
