# 10 预处理

## 预处理指令概览

| 指令 | 作用 |
|------|------|
| `#include` | 包含头文件 |
| `#define` | 宏定义 |
| `#undef` | 取消宏定义 |
| `#if` / `#ifdef` / `#ifndef` | 条件编译 |
| `#else` / `#elif` / `#endif` | 条件分支 |
| `#pragma` | 编译器相关（实现依赖） |
| `#error` | 编译报错并输出信息 |

**特点：** 以 `#` 开头，**不需要**分号结尾。

## 头文件包含

| 写法 | 搜索顺序 |
|------|----------|
| `#include <stdio.h>` | 系统目录（标准库） |
| `#include "myfile.h"` | 当前目录 → 系统目录 |

**常见标准头文件：**

| 头文件 | 内容 |
|--------|------|
| `stdio.h` | 输入输出 |
| `stdlib.h` | 内存分配、转换、exit |
| `string.h` | 字符串函数 |
| `math.h` | 数学函数 |
| `ctype.h` | 字符处理 |
| `time.h` | 时间日期 |
| `stdbool.h` | bool 类型（C99） |

## 宏定义 #define

### 常量宏

```c
#define PI 3.14159
#define MAX 100
```

### 带参宏

```c
#define MAX(a, b) ((a) > (b) ? (a) : (b))
#define SQUARE(x) ((x) * (x))
```

| 注意 | 说明 |
|------|------|
| 加括号 | 防止运算符优先级问题 |
| 副作用 | 避免 `#define SQ(x) x*x` 在 `SQ(i++)` 时出错 |
| 无类型检查 | 宏只是文本替换 |

### 宏 vs const

| | `#define` | `const` |
|--|-----------|---------|
| 处理阶段 | 预处理 | 编译 |
| 类型 | 无 | 有 |
| 调试 | 难 | 易 |
| 推荐 | 条件编译、简单常量 | 一般常量 |

## 条件编译

```c
#define DEBUG 1

#ifdef DEBUG
    printf("调试信息\n");
#endif

#ifndef HEADER_H
#define HEADER_H
/* 头文件内容 */
#endif
```

| 指令 | 含义 |
|------|------|
| `#ifdef MACRO` | 宏已定义则编译 |
| `#ifndef MACRO` | 宏未定义则编译（头文件保护） |
| `#if 表达式` | 表达式非 0 则编译 |
| `#else` | 否则分支 |
| `#endif` | 结束条件块 |

## 常用预处理技巧

| 场景 | 写法 |
|------|------|
| 头文件保护 | `#ifndef XXX_H` / `#define XXX_H` / `#endif` |
| 调试开关 | `#ifdef DEBUG` ... `#endif` |
| 跨平台 | `#ifdef _WIN32` ... `#else` ... |
| 字符串化 | `#define STR(x) #x` → `STR(hello)` → `"hello"` |
| 连接符号 | `#define CAT(a,b) a##b` |

## 预定义宏

| 宏 | 含义 |
|----|------|
| `__FILE__` | 当前源文件名 |
| `__LINE__` | 当前行号 |
| `__DATE__` | 编译日期 |
| `__TIME__` | 编译时间 |
| `__func__` | 当前函数名（C99） |

```c
printf("错误：%s 第 %d 行\n", __FILE__, __LINE__);
```

## #pragma 示例

```c
#pragma once              /* 部分编译器：头文件只包含一次 */
#pragma warning(disable:4996)  /* MSVC：忽略某警告 */
```

## 预处理执行顺序

1. 处理 `#include`、展开 `#define`
2. 处理条件编译，删除不满足条件的代码
3. 将结果交给编译器
