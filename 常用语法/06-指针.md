# 06 指针

## 指针概念

| 概念 | 说明 |
|------|------|
| 地址 | 内存中每个字节的编号 |
| 指针 | 存放地址的变量 |
| 取地址 `&` | 获取变量地址 |
| 解引用 `*` | 通过指针访问所指对象 |

## 指针声明与使用

| 写法 | 含义 |
|------|------|
| `int *p;` | p 是指向 int 的指针 |
| `p = &x;` | p 指向 x |
| `*p = 10;` | 通过 p 修改 x 的值 |
| `int *p = NULL;` | 空指针，不指向有效对象 |

```c
int x = 5;
int *p = &x;
printf("%d\n", *p);   /* 输出 5 */
*p = 10;              /* x 变为 10 */
```

## 指针与数组

| 关系 | 说明 |
|------|------|
| 数组名 | 多数情况下等价于首元素地址 |
| `a[i]` | 等价于 `*(a + i)` |
| `p[i]` | 等价于 `*(p + i)` |
| 指针移动 | `p+1` 移动一个**元素**大小，不是 1 字节 |

## 指针运算

| 运算 | 说明 |
|------|------|
| `p + n` | 向后移 n 个元素 |
| `p - n` | 向前移 n 个元素 |
| `p1 - p2` | 两指针间元素个数（同数组内） |
| `p1 == p2` | 比较地址是否相同 |

## 常见指针类型

| 类型 | 说明 |
|------|------|
| `int *p` | 指向 int |
| `char *p` | 指向字符，常用于字符串 |
| `void *p` | 通用指针，需强转后使用 |
| `int **pp` | 指向指针的指针 |
| `int (*fp)(int)` | 函数指针 |

## 指针作为函数参数

| 用途 | 示例 |
|------|------|
| 修改实参 | `void swap(int *a, int *b)` |
| 传递数组 | `void func(int *arr, int n)` |
| 返回多个结果 | 通过指针参数写回 |
| 传递大结构体 | 传地址避免复制 |

## 指针与 const

| 写法 | 含义 |
|------|------|
| `const int *p` | 指向常量的指针，*p 不可改 |
| `int *const p` | 指针常量，p 不可改指向 |
| `const int *const p` | 两者都不可改 |

## 危险用法（务必避免）

| 错误 | 后果 |
|------|------|
| 未初始化指针解引用 | 未定义行为、崩溃 |
| 返回局部变量地址 | 悬空指针 |
| 越界访问 | 数据损坏、崩溃 |
| 释放后继续使用 | 悬空指针 |
| 重复 free | 堆损坏 |

## 指针速查示例

| 需求 | 代码 |
|------|------|
| 遍历数组 | `for (int *p=a; p<a+n; p++) printf("%d", *p);` |
| 字符串遍历 | `while (*s) { ...; s++; }` |
| 动态分配 | `int *p = (int*)malloc(n * sizeof(int));` |
| 释放内存 | `free(p); p = NULL;` |

## 指针与数组作参数的区别

| 传参方式 | sizeof 在函数内 | 能否改指向 |
|----------|-----------------|------------|
| 数组 `int a[]` | 指针大小（8/4） | 形参是指针副本 |
| 指针 `int *p` | 指针大小 | 改 p 不影响实参 |

**获取数组长度：** 调用前用 `sizeof(a)/sizeof(a[0])` 计算，传入函数。
