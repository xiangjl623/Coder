# 1 程序基础

## 1.1 最小程序结构

```c
#include <stdio.h>          /* 预处理：包含头文件 */

int main(void) {            /* 主函数，程序入口 */
    printf("Hello, C!\n");
    return 0;               /* 返回 0 表示正常结束 */
}
```

## 1.2 编译与运行流程


| 阶段   | 说明                               | 常见产物                  |
| -------- | ------------------------------------ | --------------------------- |
| 预处理 | 处理`#` 指令（include、define 等） | `.i` 文件                 |
| 编译   | 将 C 代码翻译为汇编                | `.s` 文件                 |
| 汇编   | 汇编代码转为机器码                 | `.o` / `.obj` 目标文件    |
| 链接   | 合并目标文件与库，生成可执行文件   | `.exe` / 无后缀可执行文件 |

**常见命令（GCC）：**


| 命令                         | 作用                   |
| ------------------------------ | ------------------------ |
| `gcc hello.c -o hello`       | 编译并指定输出名       |
| `gcc -Wall hello.c -o hello` | 开启常见警告           |
| `gcc -c hello.c`             | 只编译不链接，生成`.o` |
| `./hello`                    | 运行（Linux/macOS）    |
| `hello.exe`                  | 运行（Windows）        |

## 1.8 C语言主要特点（速记）


| 特点           | 说明                         |
| ---------------- | ------------------------------ |
| 面向过程       | 以函数为组织单位             |
| 编译型         | 先编译再运行，效率高         |

| 无自动内存管理 | 动态内存需手动`free`         |

# 4 流程控制

## 4.1 if 语句


| 形式     | 语法                           | 说明               |
| ---------- | -------------------------------- | -------------------- |
| 单分支   | `if (条件) 语句;`              | 条件为真执行       |
| 双分支   | `if (条件) 语句1; else 语句2;` | 二选一             |
| 多分支   | `if ... else if ... else`      | 多条件             |
| 复合语句 | `if (条件) { ... }`            | 多条语句必须加`{}` |

```c
if (score >= 90)
    grade = 'A';
else if (score >= 60)
    grade = 'B';
else
    grade = 'F';
```

## 4.2 switch 语句


| 要点    | 说明                           |
| --------- | -------------------------------- |
| 表达式  | 必须是整型或枚举               |
| case    | 后接整型**常量**表达式         |
| break   | 通常每个 case 末尾加，防止贯穿 |
| default | 可选，所有 case 不匹配时执行   |

```c
switch (op) {
    case '+': result = a + b; break;
    case '-': result = a - b; break;
    default:  printf("无效\n"); break;
}
```

## 4.3 for 循环


| 部分   | 作用                     |
| -------- | -------------------------- |
| 初始化 | 循环变量初值，只执行一次 |
| 条件   | 每次循环前判断，假则退出 |
| 更新   | 每次循环体后执行         |

```c
for (int i = 0; i < 10; i++)
    printf("%d ", i);
```

## 4.4 while 循环

```c
while (条件) {
    /* 循环体 */
}
```

**特点：** 先判断后执行，条件一开始为假则一次都不执行。

int n = 100;

while（n < 50) {

//....

n = n -2;

}

int n = n + 100;

## 4.5 do-while 循环

```c
do {
    /* 循环体 */
} while (条件);   /* 注意末尾分号 */
```

**特点：** 先执行后判断，至少执行一次。

## 4.6 三种循环对比


| 循环       | 判断时机 | 最少执行次数 | 典型场景               |
| ------------ | ---------- | -------------- | ------------------------ |
| `for`      | 前       | 0            | 已知次数               |
| `while`    | 前       | 0            | 未知次数、条件驱动     |
| `do-while` | 后       | 1            | 至少执行一次（如菜单） |

## 4.7 break 与 continue


| 语句       | 作用                             | 适用                      |
| ------------ | ---------------------------------- | --------------------------- |
| `break`    | 跳出**当前**循环或 switch        | for/while/do-while/switch |
| `continue` | 跳过本次循环剩余语句，进入下一次 | for/while/do-while        |

## 4.8 goto（了解即可）

```c
goto label;
/* ... */
label:
    /* 跳转目标 */
```

**建议：** 初学少用，优先用结构化控制语句。

# 5 函数

## 5.1 函数基本结构

```c
返回类型 函数名(参数列表) {
    /* 函数体 */
    return 返回值;   /* void 函数可省略或写 return; */
}
```

## 5.2 函数声明与定义


| 概念         | 说明               | 示例                     |
| -------------- | -------------------- | -------------------------- |
| 声明（原型） | 告诉编译器函数存在 | `int max(int a, int b);` |
| 定义         | 函数的具体实现     | 含函数体                 |
| 调用         | 使用函数           | `int m = max(3, 5);`     |

## 5.3 参数传递


| 方式     | 语法                     | 特点               |
| ---------- | -------------------------- | -------------------- |
| 值传递   | `void f(int x)`          | 修改形参不影响实参 |
| 地址传递 | `void f(int *x)`         | 可通过指针修改实参 |
| 数组传递 | `void f(int a[], int n)` | 实际传首元素地址   |

```c
void swap(int *a, int *b) {
    int t = *a; *a = *b; *b = t;
}
/* 调用：swap(&x, &y); */
```

## 5.4 返回值


| 返回类型           | 说明                                 |
| -------------------- | -------------------------------------- |
| `int`、`double` 等 | `return 表达式;`                     |
| `void`             | 无返回值，`return;` 或直接结束       |
| 指针               | 可返回指针，**不要返回局部变量地址** |

## 5.5 递归函数


| 要素     | 说明                   |
| ---------- | ------------------------ |
| 终止条件 | 必须存在，否则无限递归 |
| 递归关系 | 大问题化为小问题       |

```c
int fact(int n) {
    if (n <= 1) return 1;          /* 终止条件 */
    return n * fact(n - 1);        /* 递归 */
}
```

## 5.6 存储类型


| 关键字           | 作用                         | 作用域 | 生命周期     |
| ------------------ | ------------------------------ | -------- | -------------- |
| `auto`           | 局部变量默认                 | 块内   | 函数调用期间 |
| `static`（局部） | 保留值，不重新初始化         | 块内   | 整个程序     |
| `static`（全局） | 仅本文件可见                 | 文件内 | 整个程序     |
| `extern`         | 声明外部定义的变量/函数      | 跨文件 | —           |
| `register`       | 建议放寄存器（编译器可忽略） | 块内   | 函数调用期间 |

## 5.7 函数分类速查


| 类型         | 示例                                |
| -------------- | ------------------------------------- |
| 无参无返回值 | `void menu(void);`                  |
| 有参有返回值 | `int add(int a, int b);`            |
| 数组作参数   | `void sort(int a[], int n);`        |
| 字符串作参数 | `int len(char s[]);`                |
| 函数指针参数 | `void qsort(..., int (*cmp)(...));` |

## 5.8 main 函数


| 形式                               | 说明                                |
| ------------------------------------ | ------------------------------------- |
| `int main(void)`                   | 无命令行参数                        |
| `int main(int argc, char *argv[])` | 接收命令行参数                      |
| 返回值                             | `return 0;` 表示成功，非 0 表示异常 |

## 5.9 头文件与多文件


| 文件    | 内容                   |
| --------- | ------------------------ |
| `xxx.h` | 函数声明、宏、类型定义 |
| `xxx.c` | 函数实现               |
| 使用    | `#include "xxx.h"`     |

**示例：**

```c
/* math_utils.h */
int add(int a, int b);

/* math_utils.c */
int add(int a, int b) { return a + b; }

/* main.c */
#include "math_utils.h"
int main(void) { printf("%d\n", add(1, 2)); return 0; }
```

## 5.10 内联函数（C99）

```c
inline int square(int x) { return x * x; }
```

编译器可能将函数体直接展开，减少调用开销。

# 6 指针

## 6.1 指针概念


| 概念      | 说明                 |
| ----------- | ---------------------- |
| 地址      | 内存中每个字节的编号 |
| 指针      | 存放地址的变量       |
| 取地址`&` | 获取变量地址         |
| 解引用`*` | 通过指针访问所指对象 |

## 6.2 指针声明与使用


| 写法             | 含义                   |
| ------------------ | ------------------------ |
| `int *p;`        | p 是指向 int 的指针    |
| `p = &x;`        | p 指向 x               |
| `*p = 10;`       | 通过 p 修改 x 的值     |
| `int *p = NULL;` | 空指针，不指向有效对象 |

```c
int x = 5;
int *p = &x;
printf("%d\n", *p);   /* 输出 5 */
*p = 10;              /* x 变为 10 */
```

## 6.3 指针与数组


| 关系     | 说明                                    |
| ---------- | ----------------------------------------- |
| 数组名   | 多数情况下等价于首元素地址              |
| `a[i]`   | 等价于`*(a + i)`                        |
| `p[i]`   | 等价于`*(p + i)`                        |
| 指针移动 | `p+1` 移动一个**元素**大小，不是 1 字节 |

## 6.4 指针运算


| 运算       | 说明                         |
| ------------ | ------------------------------ |
| `p + n`    | 向后移 n 个元素              |
| `p - n`    | 向前移 n 个元素              |
| `p1 - p2`  | 两指针间元素个数（同数组内） |
| `p1 == p2` | 比较地址是否相同             |

## 6.5 常见指针类型


| 类型             | 说明                   |
| ------------------ | ------------------------ |
| `int *p`         | 指向 int               |
| `char *p`        | 指向字符，常用于字符串 |
| `void *p`        | 通用指针，需强转后使用 |
| `int **pp`       | 指向指针的指针         |
| `int (*fp)(int)` | 函数指针               |

## 6.6 指针作为函数参数


| 用途         | 示例                         |
| -------------- | ------------------------------ |
| 修改实参     | `void swap(int *a, int *b)`  |
| 传递数组     | `void func(int *arr, int n)` |
| 返回多个结果 | 通过指针参数写回             |
| 传递大结构体 | 传地址避免复制               |

## 6.7 指针与 const


| 写法                 | 含义                      |
| ---------------------- | --------------------------- |
| `const int *p`       | 指向常量的指针，*p 不可改 |
| `int *const p`       | 指针常量，p 不可改指向    |
| `const int *const p` | 两者都不可改              |

## 6.8 危险用法（务必避免）


| 错误               | 后果             |
| -------------------- | ------------------ |
| 未初始化指针解引用 | 未定义行为、崩溃 |
| 返回局部变量地址   | 悬空指针         |
| 越界访问           | 数据损坏、崩溃   |
| 释放后继续使用     | 悬空指针         |
| 重复 free          | 堆损坏           |

## 6.9 指针速查示例


| 需求       | 代码                                           |
| ------------ | ------------------------------------------------ |
| 遍历数组   | `for (int *p=a; p<a+n; p++) printf("%d", *p);` |
| 字符串遍历 | `while (*s) { ...; s++; }`                     |
| 动态分配   | `int *p = (int*)malloc(n * sizeof(int));`      |
| 释放内存   | `free(p); p = NULL;`                           |

## 6.10 指针与数组作参数的区别


| 传参方式      | sizeof 在函数内 | 能否改指向      |
| --------------- | ----------------- | ----------------- |
| 数组`int a[]` | 指针大小（8/4） | 形参是指针副本  |
| 指针`int *p`  | 指针大小        | 改 p 不影响实参 |

**获取数组长度：** 调用前用 `sizeof(a)/sizeof(a[0])` 计算，传入函数。

# 7 数组

## 7.1 一维数组


| 写法       | 说明             | 示例                             |
| ------------ | ------------------ | ---------------------------------- |
| 定义       | `类型 名[长度];` | `int a[10];`                     |
| 初始化     | 定义时赋初值     | `int a[5] = {1,2,3,4,5};`        |
| 部分初始化 | 其余为 0         | `int a[5] = {1,2};` → 1,2,0,0,0 |
| 省略长度   | 由初值个数决定   | `int a[] = {1,2,3};` → 长度 3   |
| 全零       | 常用写法         | `int a[10] = {0};`               |

## 7.2 数组访问


| 规则     | 说明                         |
| ---------- | ------------------------------ |
| 下标     | 从**0** 开始                 |
| 合法范围 | `0` ~ `长度-1`               |
| 越界     | C 不检查，可能导致未定义行为 |

```c
int a[5] = {10, 20, 30, 40, 50};
printf("%d\n", a[0]);    /* 10 */
printf("%d\n", a[2]);    /* 30 */
a[4] = 100;              /* 修改最后一个元素 */
```

## 7.3 二维数组


| 写法     | 说明                     |
| ---------- | -------------------------- |
| 定义     | `类型 名[行][列];`       |
| 初始化   | 按行赋初值               |
| 元素个数 | 行 × 列                 |
| 存储     | **行优先**（先存第一行） |

```c
int a[2][3] = {{1,2,3}, {4,5,6}};
printf("%d\n", a[1][2]);   /* 6 */
```

## 7.4 数组与 sizeof


| 表达式                   | 含义（一维 int a[10]） |
| -------------------------- | ------------------------ |
| `sizeof(a)`              | 整个数组字节数（40）   |
| `sizeof(a[0])`           | 单个元素字节数（4）    |
| `sizeof(a)/sizeof(a[0])` | 元素个数（10）         |

## 7.5 数组作为函数参数

```c
void printArray(int a[], int n) {
    for (int i = 0; i < n; i++)
        printf("%d ", a[i]);
}

int main(void) {
    int arr[] = {1, 2, 3, 4, 5};
    printArray(arr, 5);
    return 0;
}
```


| 要点       | 说明                                       |
| ------------ | -------------------------------------------- |
| 传参退化   | 数组名传参变为指针                         |
| 必须传长度 | 函数内无法 sizeof 得元素个数               |
| 二维数组   | 需指定列数：`void f(int a[][4], int rows)` |

## 7.6 常见操作


| 操作   | 代码思路               |
| -------- | ------------------------ |
| 遍历   | `for (i=0; i<n; i++)`  |
| 求和   | 累加`a[i]`             |
| 求最大 | 设 max=a[0]，逐个比较  |
| 逆序   | 首尾交换，向中间靠拢   |
| 查找   | 线性遍历或二分（有序） |
| 复制   | 逐个赋值或`memcpy`     |

## 7.7 数组与指针


| 表达式     | 类型/含义        |
| ------------ | ------------------ |
| `a`        | 指向首元素的指针 |
| `a + i`    | 第 i 个元素地址  |
| `*(a + i)` | 等价`a[i]`       |
| `&a[i]`    | 第 i 个元素地址  |

## 7.8 字符数组 vs 字符指针


|        | `char s[] = "hi";` | `char *s = "hi";`  |
| -------- | -------------------- | -------------------- |
| 存储   | 栈上数组，内容可改 | 指针指向字符串常量 |
| 修改   | `s[0]='H'` 合法    | 修改内容非法       |
| sizeof | 数组大小           | 指针大小           |

# 8 字符串

## 8.1 字符串本质


| 要点   | 说明                              |
| -------- | ----------------------------------- |
| 本质   | 以`\0` 结尾的 `char` 数组         |
| 字面量 | `"Hello"` 含 6 字节：H e l l o \0 |
| 结束符 | `'\0'`，ASCII 值为 0              |
| 头文件 | `#include <string.h>`             |

## 8.2 字符串定义方式


| 写法                    | 说明                         |
| ------------------------- | ------------------------------ |
| `char s[] = "hello";`   | 字符数组，可修改             |
| `char s[20] = "hello";` | 指定大小，剩余为`\0`         |
| `char *s = "hello";`    | 指向字符串常量，**不可修改** |
| `char s[20];`           | 未初始化，需后续赋值         |

## 8.3 字符串输入


| 函数                    | 用法           | 特点                                 |
| ------------------------- | ---------------- | -------------------------------------- |
| `scanf("%s", s)`        | 读无空格字符串 | 遇空白停止，**不安全**（需控制长度） |
| `fgets(s, size, stdin)` | 读一行         | 安全，含`\n`，推荐                   |
| `gets(s)`               | 已废弃         | **禁止使用**                         |

```c
char s[100];
fgets(s, sizeof(s), stdin);
s[strcspn(s, "\n")] = '\0';   /* 去掉换行 */
```

## 8.4 字符串输出


| 函数               | 格式符/用法  |
| -------------------- | -------------- |
| `printf("%s", s)`  | 输出到`\0`   |
| `puts(s)`          | 输出并换行   |
| `fputs(s, stdout)` | 输出到指定流 |

## 8.5 string.h 常用函数


| 函数      | 原型                                           | 功能               | 返回值           |
| ----------- | ------------------------------------------------ | -------------------- | ------------------ |
| `strlen`  | `size_t strlen(const char *s)`                 | 求长度（不含`\0`） | 字符个数         |
| `strcpy`  | `char *strcpy(char *dst, const char *src)`     | 复制               | dst              |
| `strncpy` | `char *strncpy(..., size_t n)`                 | 最多复制 n 个      | dst              |
| `strcat`  | `char *strcat(char *dst, const char *src)`     | 连接               | dst              |
| `strcmp`  | `int strcmp(const char *a, const char *b)`     | 比较               | 0 相等，>0 或 <0 |
| `strchr`  | `char *strchr(const char *s, int c)`           | 找字符             | 首次出现位置     |
| `strstr`  | `char *strstr(const char *s, const char *sub)` | 找子串             | 首次出现位置     |

## 8.6 字符串比较规则


| strcmp 返回值 | 含义                   |
| --------------- | ------------------------ |
| `0`           | 两串相等               |
| `> 0`         | 第一个更大（按字典序） |
| `< 0`         | 第一个更小             |

**注意：** 不要用 `==` 比较字符串，那是比较地址。

| 函数                        | 功能        |
| ----------------------------- | ------------- |
| `isalpha(c)`                | 是否字母    |
| `isdigit(c)`                | 是否数字    |
| `isupper(c)` / `islower(c)` | 是否大/小写 |
| `isspace(c)`                | 是否空白    |
| `toupper(c)` / `tolower(c)` | 转大/小写   |

## 8.8 手动实现常用操作


| 操作   | 思路                        |
| -------- | ----------------------------- |
| 求长度 | `while (*s++) n++;`         |
| 复制   | `while ((*d++ = *s++));`    |
| 比较   | 逐字符比，遇`\0` 或不等则停 |
| 反转   | 首尾指针交换                |

## 8.9 安全注意事项


| 问题           | 建议                               |
| ---------------- | ------------------------------------ |
| 缓冲区溢出     | 用`strncpy`、`fgets`，确保目标够大 |
| strcpy 越界    | 先检查 dst 容量 ≥ src 长度+1      |
| 字符串常量修改 | `char *p = "abc"; p[0]='A';` 非法  |
| scanf %s       | 限制宽度：`scanf("%99s", s)`       |

## 9.1 结构体 struct

### 9.1.1 定义与使用

```c
struct Student {
```

| 操作     | 语法                                      |
| ---------- | ------------------------------------------- |

| 初始化   | `struct Student s = {1001, "Tom", 90.5};` |
| 访问成员 | `s.id`、`s.name`                          |
| 指针访问 | `p->id` 等价 `(*p).id`                    |

### 9.1.2 typedef 简化

```c
typedef struct {

```

## 9.2 结构体数组

```c
struct Student st[3];
st[0].id = 1;
/* 或初始化 */
struct Student st[] = {
    {1, "A", 80},
    {2, "B", 90}
};
```

## 9.3 结构体作为函数参数


| 方式     | 特点                              |
| ---------- | ----------------------------------- |
| 值传递   | 复制整个结构体，大结构体效率低    |
| 指针传递 | `void f(struct Student *p)`，推荐 |

## 9.4 结构体对齐（了解）


| 概念   | 说明                   |
| -------- | ------------------------ |
| 对齐   | 成员按一定字节边界存放 |
| sizeof | 可能大于各成员之和     |
| 原因   | CPU 访问对齐地址更高效 |

## 9.5 共用体 union


| 特点 | 说明                           |
| ------ | -------------------------------- |
| 内存 | 所有成员**共享**同一块内存     |
| 大小 | 等于最大成员的大小             |
| 用途 | 节省空间、多种格式解释同一数据 |

```c
union Data {
    int i;
    float f;
    char c;
};
union Data d;
d.i = 10;      /* 此时用 i */
d.f = 3.14f;   /* 覆盖 i 的内容 */
```

## 9.6 枚举 enum

```c
enum Color { RED, GREEN, BLUE };
enum Color c = GREEN;   /* GREEN 值为 1 */
```


| 特点   | 说明                              |
| -------- | ----------------------------------- |
| 默认值 | 从 0 递增                         |
| 可指定 | `enum { A=1, B, C };` → B=2, C=3 |
| 本质   | 整型常量                          |

## 9.7 位域（节省空间）

```c
struct Flags {
    unsigned int a : 1;   /* 占 1 位 */
    unsigned int b : 3;   /* 占 3 位 */
};
```

## 9.8 对比速查


|          | struct               | union           | enum             |
| ---------- | ---------------------- | ----------------- | ------------------ |
| 成员共存 | 是                   | 否（共享内存）  | —               |
| 主要用途 | 组合不同类型数据     | 多种视图/省空间 | 命名整型常量     |
| 大小     | 各成员之和（含对齐） | 最大成员        | 通常 sizeof(int) |

## 9.9 嵌套结构体

```c
typedef struct {
    int year, month, day;
} Date;

typedef struct {
    char name[32];
    Date birthday;
} Person;

Person p;

```

## 10.3 宏定义 #define

### 10.3.1 常量宏

```c

```

### 10.3.2 带参宏

```c
#define MAX(a, b) ((a) > (b) ? (a) : (b))

```


| 注意       | 说明                                        |
| ------------ | --------------------------------------------- |
| 加括号     | 防止运算符优先级问题                        |
| 副作用     | 避免`#define SQ(x) x*x` 在 `SQ(i++)` 时出错 |

### 10.3.3 宏 vs const


|          | `#define`          | `const`  |
| ---------- | -------------------- | ---------- |
| 处理阶段 | 预处理             | 编译     |
| 类型     | 无                 | 有       |
| 调试     | 难                 | 易       |
| 推荐     | 条件编译、简单常量 | 一般常量 |

## 10.4 条件编译

```c
    printf("调试信息\n");

```


| 指令            | 含义                         |
| ----------------- | ------------------------------ |
| `#ifdef MACRO`  | 宏已定义则编译               |
| `#ifndef MACRO` | 宏未定义则编译（头文件保护） |
| `#if 表达式`    | 表达式非 0 则编译            |
| `#else`         | 否则分支                     |
| `#endif`        | 结束条件块                   |

## 10.5 常用预处理技巧


| 场景       | 写法                                             |

| 头文件保护 | `#ifndef XXX_H` / `#define XXX_H` / `#endif`     |
| 调试开关   | `#ifdef DEBUG` ... `#endif`                      |
| 跨平台     | `#ifdef _WIN32` ... `#else` ...                  |
| 字符串化   | `#define STR(x) #x` → `STR(hello)` → `"hello"` |
| 连接符号   | `#define CAT(a,b) a##b`                          |

## 10.6 预定义宏

| `__FILE__` | 当前源文件名      |

# 13 动态内存

头文件：`#include <stdlib.h>`

## 13.1 为什么需要动态内存


| 对比     | 栈（自动变量/数组） | 堆（动态分配）       |
| ---------- | --------------------- | ---------------------- |
| 大小     | 编译期或固定        | 运行期按需           |
| 生命周期 | 函数结束释放        | 手动 free 前一直存在 |
| 典型用途 | 局部变量            | 链表、可变长数据     |

## 13.2 四个核心函数


| 函数                  | 功能                              | 返回值           |
| ----------------------- | ----------------------------------- | ------------------ |
| `malloc(size)`        | 分配 size 字节，**不初始化**      | void*，失败 NULL |
| `calloc(n, size)`     | 分配 n×size 字节，**初始化为 0** | void*            |
| `realloc(p, newSize)` | 调整已分配块大小                  | void*，失败 NULL |
| `free(p)`             | 释放 p 指向的内存                 | void             |

## 13.3 基本用法

```c
#include <stdlib.h>

/* 分配 10 个 int */
int *arr = (int *)malloc(10 * sizeof(int));
if (arr == NULL) {
    /* 分配失败处理 */
    return 1;
}

/* 使用 arr[0] ~ arr[9] */

free(arr);       /* 释放 */
arr = NULL;      /* 避免悬空指针 */
```

## 13.4 函数对比


|        | malloc                    | calloc                   |
| -------- | --------------------------- | -------------------------- |
| 参数   | 总字节数                  | 元素个数 + 元素大小      |
| 初始化 | 不初始化（内容不确定）    | 全部置 0                 |
| 示例   | `malloc(n * sizeof(int))` | `calloc(n, sizeof(int))` |

## 13.5 realloc 用法

```c
int *p = malloc(5 * sizeof(int));
/* 需要更大空间 */
int *q = realloc(p, 10 * sizeof(int));
if (q == NULL) {
    free(p);    /* 失败时原块仍有效 */
    return 1;
}
p = q;
```


| 注意     | 说明                    |
| ---------- | ------------------------- |
| 可能移动 | 返回地址可能与原 p 不同 |
| 失败     | 原内存块仍然有效        |
| 缩小     | 多余部分被释放          |

## 13.6 动态二维数组

```c
int rows = 3, cols = 4;
int **a = (int **)malloc(rows * sizeof(int *));
for (int i = 0; i < rows; i++)
    a[i] = (int *)malloc(cols * sizeof(int));

/* 使用 a[i][j] */

for (int i = 0; i < rows; i++) free(a[i]);
free(a);
```

## 13.7 常见错误


| 错误            | 后果       | 正确做法              |
| ----------------- | ------------ | ----------------------- |
| 不检查 NULL     | 解引用崩溃 | `if (p == NULL)`      |
| 忘记 free       | 内存泄漏   | 每个 malloc 对应 free |
| 重复 free       | 堆损坏     | free 后置 NULL        |
| free 后继续使用 | 悬空指针   | free 后不再访问       |
| 越界写入        | 堆损坏     | 严格控制在分配大小内  |

## 13.8 内存泄漏

**定义：** 动态分配的内存未释放且无法再访问。

**预防：**

- 谁分配谁释放，或约定清晰的所有权
- 函数返回前 free 局部分配
- 复杂结构（链表）遍历释放每个结点

## 13.9 链表结点分配示例

```c
typedef struct Node {
    int data;
    struct Node *next;
} Node;

Node *p = (Node *)malloc(sizeof(Node));
p->data = 10;
p->next = NULL;
/* 不用时 */
free(p);
```

## 13.10 malloc 与 sizeof


| 推荐写法                  | 原因             |
| --------------------------- | ------------------ |
| `malloc(n * sizeof(int))` | 类型变更时仍正确 |
| 避免`malloc(100)`         | 魔法数字，易错   |

## 13.11 栈 vs 堆 速查


| 操作     | 栈                | 堆                 |
| ---------- | ------------------- | -------------------- |
| 分配     | 自动              | malloc/calloc      |
| 释放     | 自动              | free               |
| 大小限制 | 较小（通常几 MB） | 较大（受系统限制） |
| 速度     | 快                | 相对慢             |
| 碎片     | 无                | 可能产生           |
