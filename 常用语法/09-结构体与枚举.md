# 09 结构体与枚举

## 结构体 struct

### 定义与使用

```c
struct Student {
    int id;
    char name[32];
    float score;
};

struct Student s1;
s1.id = 1001;
strcpy(s1.name, "Tom");
s1.score = 90.5;
```

| 操作 | 语法 |
|------|------|
| 定义变量 | `struct Student s;` |
| 初始化 | `struct Student s = {1001, "Tom", 90.5};` |
| 访问成员 | `s.id`、`s.name` |
| 指针访问 | `p->id` 等价 `(*p).id` |

### typedef 简化

```c
typedef struct {
    int x, y;
} Point;

Point p = {10, 20};   /* 无需写 struct */
```

## 结构体数组

```c
struct Student st[3];
st[0].id = 1;
/* 或初始化 */
struct Student st[] = {
    {1, "A", 80},
    {2, "B", 90}
};
```

## 结构体作为函数参数

| 方式 | 特点 |
|------|------|
| 值传递 | 复制整个结构体，大结构体效率低 |
| 指针传递 | `void f(struct Student *p)`，推荐 |

## 结构体对齐（了解）

| 概念 | 说明 |
|------|------|
| 对齐 | 成员按一定字节边界存放 |
| sizeof | 可能大于各成员之和 |
| 原因 | CPU 访问对齐地址更高效 |

## 共用体 union

| 特点 | 说明 |
|------|------|
| 内存 | 所有成员**共享**同一块内存 |
| 大小 | 等于最大成员的大小 |
| 用途 | 节省空间、多种格式解释同一数据 |

```c
union Data {
    int i;
    float f;
    char c;
};
union Data d;
d.i = 10;      /* 此时用 i */
d.f = 3.14f;   /* 覆盖 i 的内容 */
```

## 枚举 enum

```c
enum Color { RED, GREEN, BLUE };
enum Color c = GREEN;   /* GREEN 值为 1 */
```

| 特点 | 说明 |
|------|------|
| 默认值 | 从 0 递增 |
| 可指定 | `enum { A=1, B, C };` → B=2, C=3 |
| 本质 | 整型常量 |

## 位域（节省空间）

```c
struct Flags {
    unsigned int a : 1;   /* 占 1 位 */
    unsigned int b : 3;   /* 占 3 位 */
};
```

## 对比速查

| | struct | union | enum |
|--|--------|-------|------|
| 成员共存 | 是 | 否（共享内存） | — |
| 主要用途 | 组合不同类型数据 | 多种视图/省空间 | 命名整型常量 |
| 大小 | 各成员之和（含对齐） | 最大成员 | 通常 sizeof(int) |

## 嵌套结构体

```c
typedef struct {
    int year, month, day;
} Date;

typedef struct {
    char name[32];
    Date birthday;
} Person;

Person p;
p.birthday.year = 2000;
```
