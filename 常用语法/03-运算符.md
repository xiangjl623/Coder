# 03 运算符

## 算术运算符

| 运算符 | 含义 | 示例 | 注意 |
|--------|------|------|------|
| `+` | 加 | `a + b` | — |
| `-` | 减 | `a - b` | — |
| `*` | 乘 | `a * b` | — |
| `/` | 除 | `a / b` | 整数/整数 → 整数（截断） |
| `%` | 取模 | `a % b` | 仅用于整型 |

**整数除法示例：** `5 / 2 = 2`，`5.0 / 2 = 2.5`

## 关系运算符

| 运算符 | 含义 | 结果类型 |
|--------|------|----------|
| `>` `<` `>=` `<=` | 比较大小 | 真为 1，假为 0 |
| `==` | 等于 | 注意与赋值 `=` 区分 |
| `!=` | 不等于 | — |

## 逻辑运算符

| 运算符 | 含义 | 示例 | 短路 |
|--------|------|------|------|
| `&&` | 逻辑与 | `a && b` | 左假则不算右 |
| `\|\|` | 逻辑或 | `a \|\| b` | 左真则不算右 |
| `!` | 逻辑非 | `!flag` | — |

## 赋值运算符

| 运算符 | 等价于 | 示例 |
|--------|--------|------|
| `=` | — | `a = 10` |
| `+=` | `a = a + b` | `a += 5` |
| `-=` | `a = a - b` | `a -= 3` |
| `*=` | `a = a * b` | `a *= 2` |
| `/=` | `a = a / b` | `a /= 2` |
| `%=` | `a = a % b` | `a %= 3` |

**结合性：** 从右到左，如 `a = b = 5` 先给 b 赋 5，再给 a 赋 5。

## 自增自减

| 写法 | 含义 | 表达式值 |
|------|------|----------|
| `++i` | i 先加 1 | 加之后的值 |
| `i++` | i 后加 1 | 加之前的值 |
| `--i` / `i--` | 同理减 1 | 同理 |

## 位运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `&` | 按位与 | `a & b` |
| `\|` | 按位或 | `a \| b` |
| `^` | 按位异或 | `a ^ b` |
| `~` | 按位取反 | `~a` |
| `<<` | 左移 | `a << 2` 相当于 ×4 |
| `>>` | 右移 | `a >> 1` 相当于 ÷2 |

## 其他运算符

| 运算符 | 名称 | 示例 |
|--------|------|------|
| `? :` | 条件运算符 | `max = a > b ? a : b` |
| `,` | 逗号 | `x = 1, y = 2`（值为最后一个） |
| `&` | 取地址 | `&x` |
| `*` | 解引用 | `*p` |
| `sizeof` | 求大小 | `sizeof(int)` |
| `.` | 结构体成员 | `stu.name` |
| `->` | 指针访问成员 | `p->name` |

## 运算符优先级（从高到低，常用部分）

| 优先级 | 运算符 | 结合性 |
|--------|--------|--------|
| 1 | `()` `[]` `->` `.` | 左→右 |
| 2 | `!` `~` `++` `--` `*` `&` `sizeof` | 右→左 |
| 3 | `*` `/` `%` | 左→右 |
| 4 | `+` `-` | 左→右 |
| 5 | `<<` `>>` | 左→右 |
| 6 | `<` `<=` `>` `>=` | 左→右 |
| 7 | `==` `!=` | 左→右 |
| 8 | `&` | 左→右 |
| 9 | `^` | 左→右 |
| 10 | `\|` | 左→右 |
| 11 | `&&` | 左→右 |
| 12 | `\|\|` | 左→右 |
| 13 | `?:` | 右→左 |
| 14 | `=` `+=` `-=` … | 右→左 |
| 15 | `,` | 左→右 |

**记忆口诀：** 单目 → 算术 → 移位 → 关系 → 位运算 → 逻辑 → 条件 → 赋值 → 逗号

## 常见表达式写法

| 场景 | 写法 |
|------|------|
| 交换两数（需临时变量） | `t=a; a=b; b=t;` |
| 求绝对值 | `x = x < 0 ? -x : x` |
| 判断奇偶 | `n % 2 == 0` |
| 判断闰年 | `(y%4==0 && y%100!=0) \|\| y%400==0` |
| 判断字符是数字 | `ch >= '0' && ch <= '9'` |
