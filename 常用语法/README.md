## 使用建议

1. **先通读 01～04**：建立程序结构与基本语法框架。
2. **重点掌握 06～08**：指针、数组、字符串是 C 语言核心。
3. **按需查阅 09～14**：做项目或写题时再查结构体、文件、内存等。
4. 表中示例可直接复制到 `.c` 文件中编译运行。
