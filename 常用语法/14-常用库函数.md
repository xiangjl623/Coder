# 14 常用库函数

## stdio.h — 输入输出

| 函数 | 功能 |
|------|------|
| `printf` / `scanf` | 格式化标准 I/O |
| `fprintf` / `fscanf` | 格式化文件 I/O |
| `sprintf` / `sscanf` | 字符串格式化 |
| `getchar` / `putchar` | 字符 I/O |
| `fgets` / `fputs` | 行 I/O |
| `fopen` / `fclose` | 打开/关闭文件 |
| `fgetc` / `fputc` | 文件字符 I/O |
| `fread` / `fwrite` | 块读写 |
| `fseek` / `ftell` / `rewind` | 文件定位 |
| `feof` / `ferror` | 文件状态 |
| `perror` | 输出系统错误信息 |

## stdlib.h — 通用工具

| 函数 | 功能 | 示例 |
|------|------|------|
| `malloc` / `calloc` / `realloc` / `free` | 动态内存 | 见 13-动态内存 |
| `exit(status)` | 立即终止程序 | `exit(0);` |
| `abort()` | 异常终止 | — |
| `atoi(s)` | 字符串→int | `atoi("123")` → 123 |
| `atof(s)` | 字符串→double | — |
| `atol(s)` | 字符串→long | — |
| `rand()` | 伪随机数 0~RAND_MAX | — |
| `srand(seed)` | 设置随机种子 | `srand(time(NULL));` |
| `abs(n)` | 整数绝对值 | — |
| `qsort` | 快速排序 | 需比较函数 |
| `bsearch` | 二分查找 | 数组须有序 |
| `system(cmd)` | 执行系统命令 | `system("cls");` |

### qsort 示例

```c
int cmp(const void *a, const void *b) {
    return *(int *)a - *(int *)b;
}
qsort(arr, n, sizeof(int), cmp);
```

|------|------|

## time.h — 时间

| 函数/类型 | 功能 |
|-----------|------|
| `time_t` | 时间类型 |
| `time(NULL)` | 当前时间戳（秒） |
| `clock()` | CPU 时钟 ticks |
| `CLOCKS_PER_SEC` | 每秒 ticks 数 |

```c
#include <time.h>
srand((unsigned)time(NULL));   /* 随机种子 */
clock_t t = clock();
/* 代码 */
printf("耗时:%f秒\n", (double)(clock()-t)/CLOCKS_PER_SEC);
```

## assert.h — 断言

```c
#include <assert.h>
assert(p != NULL);   /* 条件为假则 abort 并输出信息 */
```

## stdbool.h — 布尔（C99）

```c
#include <stdbool.h>
bool flag = true;    /* 等价 _Bool，值为 0 或 1 */
```

## 安全替代建议

| 不推荐 | 推荐 |
|--------|------|
| `gets` | `fgets` |
| `strcpy`（未知长度） | `strncpy` / `snprintf` |
| `sprintf`（无边界） | `snprintf` |
| `scanf("%s")` | `scanf("%99s", s)` 或 `fgets` |
